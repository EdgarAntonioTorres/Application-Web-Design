## What is Markdown used for?

Markdown is a lightweight markup language used to format plain text.  
It allows users to easily create structured documents using simple symbols
instead of complex code.

Markdown is commonly used for:
- Writing README files on GitHub
- Creating documentation
- Formatting text in forums and blogs
- Taking notes in a clean and readable way

One of its main advantages is that it is easy to read even without rendering,
and it can be converted into HTML and other formats.
